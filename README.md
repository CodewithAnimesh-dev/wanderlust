# 🌍 Wanderlust

A full-stack travel accommodation web application inspired by Airbnb, where users can explore destinations, create listings, upload images, leave reviews, and discover places around the world.

---

## 🚀 Live Demo

🔗 https://wanderlust-awfd.onrender.com

---

## ✨ Features

- 🔐 User Authentication (Sign Up & Login)
- 🏡 Create, Edit & Delete Listings
- ☁️ Image Upload using Cloudinary
- 🗺️ Interactive Maps with Mapbox
- 🔍 Search by Title, Location & Country
- 🏷️ Category-Based Filtering
- ⭐ Reviews & Ratings
- 🌙 Dark Mode
- 💰 Display Total Price After Taxes
- 📱 Responsive Design
- ☁️ MongoDB Atlas Database
- 🚀 Deployed on Render

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/CodewithAnimesh-dev/wanderlust.git
```

Go to project folder

```bash
cd wanderlust
```

Install dependencies

```bash
npm install
```

Create a `.env` file and add

```
ATLASDB_URL=
SECRET=
MAP_TOKEN=
CLOUD_NAME=
CLOUD_API_KEY=
CLOUD_API_SECRET=
```

Run the application

```bash
nodemon app.js
```
